# Scooter-UIR

Elyra - Electric Scooter Sharing App for Campus Mobility

## Description

Elyra is a native iOS app built with Swift and SwiftUI that provides an electric scooter sharing service for campus mobility. The app features a modern, dark-themed UI with a welcome screen showcasing the service.

## Features

- Welcome screen with Elyra branding
- Dark theme UI
- Custom scooter visualization
- Gradient effects
- French language support

## Project Structure

```
iOS/
├── Elyra/
│   ├── ElyraApp.swift          # App entry point
│   ├── Views/
│   │   └── WelcomeScreen.swift # Welcome screen view
│   ├── Assets.xcassets/        # App icons and assets
│   └── Info.plist              # App configuration
└── Elyra.xcodeproj/            # Xcode project file
```

## Requirements

- Xcode 15.0 or later
- iOS 17.0 or later
- macOS 13.0 or later

## Getting Started

1. Open the project in Xcode:
   ```bash
   open iOS/Elyra.xcodeproj
   ```

2. Select a simulator or connected device

3. Press ⌘R to build and run
