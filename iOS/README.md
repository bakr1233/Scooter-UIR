# Elyra iOS App

Native iOS app built with Swift and SwiftUI.

## Project Structure

```
iOS/
├── Elyra/
│   ├── ElyraApp.swift          # App entry point
│   ├── Views/
│   │   └── WelcomeScreen.swift # Welcome screen view
│   ├── Assets.xcassets/        # App icons and assets
│   └── Info.plist              # App configuration
└── Elyra.xcodeproj/            # Xcode project file
```

## Opening the Project

1. Open Xcode
2. File → Open
3. Navigate to `/Users/bakrbouhaya/Scooter_App/iOS/Elyra.xcodeproj`
4. Select a simulator or connected device
5. Press ⌘R to build and run

## Requirements

- Xcode 15.0 or later
- iOS 17.0 or later
- macOS 13.0 or later

## Features

- Welcome screen matching the design
- Dark theme UI
- Gradient effects
- Custom scooter visualization
- Status bar simulation
